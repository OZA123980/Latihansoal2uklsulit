# Latihansoal2uklsulit
Latihan UKL 3
Program ini menjumlahkan dua buah matriks yang dimasukkan oleh pengguna.Setiap elemen pada posisi yang sama dari kedua matriks dijumlahkan dan disimpan ke dalam matriks hasil.
Cara kerjanya adalah:
-Pengguna memasukkan jumlah baris dan kolom matriks.
-Pengguna mengisi nilai-nilai untuk matriks A dan matriks B.
-Program melakukan perulangan untuk menjumlahkan setiap elemen.
-Hasil penjumlahan disimpan dalam matriks C.
-Program menampilkan matriks hasil (A+B) ke layar.
